Files Included (8 Java Source Files)

FactorialRecursion.java – Calculates the factorial of a number using recursion.
SubsetsRecursion.java – Generates all possible subsets of a given set using recursion.
FirstNPrimesLoop.java – Generates the first n prime numbers using looping.
GCDRecursion.java – Finds the greatest common divisor (GCD) of two numbers using recursion.
LCMLoop.java – Finds the least common multiple (LCM) of two numbers using looping.
