
Basics Java Assignment – Array & Loop Tasks
Prepared for: Arumugam B

Files included (5 Java source files):

1. SumArray.java – Finds the sum of all elements in an integer array.
2. LargestElement.java – Finds the largest element in an integer array.
3. AverageArray.java – Computes the average of all elements in a double array.
4. ReverseStringArray.java – Prints elements of a string array in reverse order.
5. Fibonacci10.java – Prints the first 10 numbers of the Fibonacci sequence.

How to compile all files:
javac *.java

How to run an example:
java SumArray
java LargestElement
java Fibonacci

Notes:
- All programs use standard Java loops for array manipulation.
- You can modify array values directly in the code to test different cases.
- Programs demonstrate basic array operations and sequence generation.
