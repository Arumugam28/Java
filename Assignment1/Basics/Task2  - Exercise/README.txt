
Basics Java Assignment – All 17 Tasks
Prepared for: Arumugam B

Files included (17 Java source files):

1. HelloWorld.java – Prints “Hello, world!” to the console.
2. GreetingUser.java – Asks the user for their name and greets them.
3. CompareNumbers.java – Compares two numbers and prints greater, lesser, or equal.
4. NumberCheck.java – Checks if a number is positive, negative, or zero.
5. CheckAdult.java – Checks if user age is 18 or older.
6. CelsiusToFahrenheit.java – Converts Celsius temperature to Fahrenheit.
7. GuessNumber1to10.java – Random number guessing game (1–10).
8. RangeChecker.java – Checks if a number is in the lower or upper half (1–100).
9. CheckJavaString.java – Checks if a string contains “Java”.
10. SumOfTwoNumbers.java – Calculates sum of two numbers.
11. GuessNumber1to100.java – Random number guessing game (1–100).
12. ReverseString.java – Reverses a given string.
13. AverageCalculator.java – Reads numbers from user and calculates average.
14. PalindromeChecker.java – Checks if a string is palindrome.
15. FindMinMax.java – Finds smallest and largest numbers in an array.
16. SortNames.java – Sorts a list of names alphabetically.
17. MultiplicationTable.java – Generates multiplication table for a given number.

How to compile all files:
javac *.java
or if you have a folder named 'basics':
javac basics/*.java

How to run an example:
java HelloWorld
java GreetingUser
java GuessNumber1to100

Notes:
- All programs use java.util.Scanner for console input.
- For guessing games, enter integer guesses as prompted.
- Programs handle basic validation where required.
- You can modify variable values in the code to test different cases.
